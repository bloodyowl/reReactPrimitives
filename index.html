A collection of UI primitive blocks to build apps for the web with Reason

Demos

  • ActivityIndicator Animated loader
  • Button Simple opinionated button
  • Confirm Confirmation dialog
  • Dialog Full viewport dialog
  • FixedCollectionView A functor to build big typed collection views
  • Image Display images in a fixed space
  • ListView Render a list of items
  • Picker Equivalent of an HTML select with more rendering options
  • SectionedListView Render a list of items grouped by section
  • Sprite Functor that generates a sprite component for a given image
  • Switch Boolean input
  • TabbedView Horizontal or vertical tabs
  • Tag Tag & action button to remove it
  • TextInput Component that renders a text entry (possibly multiline & autozesized)
  • Tooltip Render a positionned tooltip on hovering
  • TouchableHighlight Click area that renders an underlay on press
  • TouchableOpacity Click area that lowers its opacity on press
